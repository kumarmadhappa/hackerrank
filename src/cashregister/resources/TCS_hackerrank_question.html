Create a cash register for a fruit store that enables to complete purchases (applying offers, if any), etc.
 

Project Setup


Classes: 

 

Register: Singleton class 

 

  • Maintains a list of items for sale and their prices.
  • Initiates and completes a checkout by returning a bill.

 
Methods:

 

  • Void startCheckout()
  • Bill finishCheckout() 
  • Double scanItem (String item) returns InvalidItemScan.
  • Register getInstance() returns the singleton instance of a register.
  • Void addItems (Map itemPriceMap) 

 
Notes: 

  • Checkout begins when a customer brings items to the billing desk and ends when the customer leaves the store after paying.
  • During checkout, the order in which items are scanned does not matter.

 

The register contains the list of items and their prices. In this exercise, the list of items and their prices are:

 

Item      | Price
-------------------------
Apple    | 2.0
-------------------------
Orange  | 1.5
-------------------------
Mango  | 1.2
-------------------------
Grapes   | 1.0
-------------------------
  
Bill: 

 

  • List of items purchased by the customer, and the price paid.
  • ArrayList of items purchased by the customer.
  • totalPrice paid by the customer.

 
Methods:

 

  • Double getTotalBill() returns the total price a customer has to pay.
  • Use addToBill (double price) to add to the bill.

 
Exceptions:

 

InvalidItemScan- The Register method returns this exception in certain cases. For example, when an item that is not part of the store inventory is scanned by the register.


Notes:

  • The Register class is a singleton class.
  • Register getInstance() returns the singleton instance of a register.
  • Use the getInstance() method to create a Register object.

 

Sample Input:

java Register "apples" 30 "orange" 10 "mango" 20

 

Sample Output:

99.0
